# Modern Art Gallery

Mini site responsive desarrollado con **HTML, SASS y Vite**.

Proyecto realizado como tercera tarea entregable de CSS, maquetado siguiendo el diseño proporcionado en Figma.

---

##  Tecnologías utilizadas

- HTML5
- SASS (SCSS)
- Vite
- Mobile First
- CSS Grid
- Flexbox
- SVG Icons

---

##  Responsive

El proyecto está desarrollado siguiendo la metodología **Mobile First**:

-  Mobile
-  Tablet (768px)
-  Desktop (1200px+)

Se utilizan media queries mediante mixins en SASS.
